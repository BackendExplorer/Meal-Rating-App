## <a id="サービスの特徴・開発の目的"></a> 📝 サービスの特徴・開発の目的

<br>

###  サービスの全体像

- このプロジェクトは、**食事のレビューを投稿してシェアできるサービス**です。

- ユーザーはログイン後、食事の写真とコメントを投稿したり、他の投稿に星評価を付けたりできます。


<br>

###  できること

<div style="height:8px;"></div>

- **食事投稿**  

  料理名、画像、国名、説明などを入力し、食事情報を投稿できます。

- **食事一覧の閲覧**
  
  投稿されたすべての食事が一覧で表示され、誰でも閲覧可能です。

- **星評価の投稿**  

  他のユーザーが投稿した食事に対して、0〜5の星評価をつけることができます。

- **ユーザー認証**

  サインアップ・ログイン機能により、ユーザー管理を実現しています。


<br>


###  作成のきっかけ

<div style="height:8px;"></div>

1. **課題意識**

   自分や他人が食べた料理の記録・共有・評価を、シンプルかつ視覚的に行える仕組みがほしいと感じたため。     

2. **解決アプローチ**

  Djangoを活用し、ログイン機能・画像付き投稿・星評価・投稿一覧表示など、基本機能を一から実装。フォーム処理やバリデーションにも対応。
    
3. **得られた学び**

   Djangoフレームワークの構造理解、フォームとモデルの連携、ログイン管理、テンプレートエンジンの使い方、基本的なWebアプリ構築スキル。
